# Changelog

## 0.2.0 - 2026-07-22

- Replaced the single-file prototype with an installable package.
- Added Git-aware file discovery and `.gitignore` support.
- Added line and column locations, masked values, and stable fingerprints.
- Added configurable checks, exclusions, allow rules, and failure policy.
- Added text, JSON, and SARIF reports.
- Added explicit scan errors, binary detection, and symlink protection.
- Added unit tests and a Python-version CI matrix.
- Rewrote the documentation around actual behaviour and limitations.
